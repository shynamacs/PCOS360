# PCOS360

PCOS360 is a mobile-first pilot MVP for women managing Polycystic Ovary Syndrome. It combines symptom tracking, nutrition planning, weight and cycle monitoring, AI coaching, and progress analytics.

## Tech stack

- Frontend: React Native + Expo + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: MongoDB
- AI: OpenAI API (stubbed in MVP)

## Structure

- `frontend/` — Expo app with onboarding, dashboard, trackers, and AI coach
- `backend/` — Express API for auth, onboarding, dashboard data, and AI coach backend

## Quick start

1. Install dependencies:
   ```bash
   yarn install
   ```
2. Start the frontend:
   ```bash
   yarn workspace frontend start
   ```
3. Start the backend:
   ```bash
   yarn workspace backend dev
   ```

## MVP scope

Implemented for pilot users:

- Authentication
- Onboarding
- Dashboard
- Symptom tracker
- Weight tracker
- Period tracker
- African meal planner
- Water tracker
- AI coach
- Progress charts
